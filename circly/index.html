Programs: Circly

About

Circly distorts images in an aesthetically-pleasing-to-me way. Little circly fractalish fragments of the image lace-likely overlay other fragments.

It started out as a little Python script that uses the Pillow image library. But I thought it might be fun to watch it do its thing as an animation, and thus this JS-powered web page.

Its algorithm is:

  • You start with an original image and a blank drawing space. Choose a big circle size.
  • Repeat:
    • Is the circle size now absurdly small? If so, stop.
    • Choose a point in the drawing space. Does it now have a color quite different than the original image? If not, move on. But if so, draw a circle using the current circle size in the color from the original image.
    • Have you drawn many circles at the current circle size? If so, reduce the circle size a skoonch.
